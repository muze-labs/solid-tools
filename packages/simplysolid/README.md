# @muze-labs/simplysolid

SimplyFlow runtime extension that exposes Solid collections through `app.solid`.

Milestone status: Milestone 6 setup conventions are implemented.

SimplySolid creates app-facing collection handles and setup status on top of Solid Workspace. It keeps Solid concerns inside `app.solid`, while Lading, Metro-OLDM/OLDM, and oldm-shape keep their own lower-level responsibilities.
